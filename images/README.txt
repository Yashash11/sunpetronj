DROP YOUR OWN PHOTOS HERE
=========================

This folder is where your real photos live. To use one of your own pictures
instead of a stock photo:

1. Save your photo into this /images folder (e.g. images/storefront.jpg).
2. Open index.html and find the image you want to replace. Every image has a
   comment right above it telling you what it is and the best size, e.g.:
       <!-- REPLACE: storefront / canopy at golden hour · 2000x1200 -->
3. Change the src to point at your file:
       src="images/storefront.jpg"
4. Update the alt="..." text to describe your photo (good for SEO + accessibility).

RECOMMENDED SIZES
-----------------
- Hero slides ............ 2000 x 1200 px (landscape, warm-toned)
- Store feature photo .... 1200 x 900 px
- Category / service cards  800 x 600 px
- Gallery photos ......... 800 x 800 px (square)
- Our Story photo ........ 1000 x 1100 px (portrait)
- Community banner ....... 1600 x 700 px (wide)

TIPS
----
- Use JPG for photos (smaller files). Keep each under ~400 KB if you can.
- Warm, bright, red-and-white-friendly shots look best.
- If an image ever fails to load, the site automatically shows a clean
  red placeholder tile with the description — never a broken image.
